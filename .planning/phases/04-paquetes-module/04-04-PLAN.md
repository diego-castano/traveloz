Create the Precios tab with real-time pricing that recomputes neto from live service assignments.

1. **Props:** `{ paquete: Paquete }`.

2. **Hooks:**
   - `usePaqueteServices(paquete.id)` -- returns junction records (aereos, alojamientos, traslados, seguros, circuitos).
   - `useServiceState()` from ServiceProvider -- access raw arrays: `state.aereos`, `state.preciosAereo`, `state.alojamientos`, `state.preciosAlojamiento`, `state.traslados`, `state.seguros`, `state.circuitos`, `state.preciosCircuito`.
   - `usePackageActions()` for `updatePaquete`.
   - `useAuth()` for `canSeePricing` and `canEdit`.
   - `useToast()` for success feedback.

3. **Live neto calculation via useMemo:**
   Recompute neto from current service assignments (NOT from `paquete.netoCalculado` which may be stale). Use `calcularNeto()` from utils:

   ```typescript
   const liveNeto = useMemo(() => {
     const assignedAereos = services.aereos.map(pa => {
       const aereo = serviceState.aereos.find(a => a.id === pa.aereoId)!;
       const precioAereo = serviceState.preciosAereo.find(p => p.id === pa.precioAereoId);
       return { aereo, precioAereo };
     });
     const assignedAlojamientos = services.alojamientos.map(pa => {
       const alojamiento = serviceState.alojamientos.find(a => a.id === pa.alojamientoId)!;
       const precioAlojamiento = serviceState.preciosAlojamiento.find(p => p.id === pa.precioAlojamientoId);
       return { alojamiento, precioAlojamiento, nochesEnEste: pa.nochesEnEste ?? undefined };
     });
     const assignedTraslados = services.traslados.map(pt => serviceState.traslados.find(t => t.id === pt.trasladoId)!);
     const assignedSeguros = services.seguros.map(ps => {
       const seguro = serviceState.seguros.find(s => s.id === ps.seguroId)!;
       return { seguro, diasCobertura: ps.diasCobertura ?? undefined };
     });
     const assignedCircuitos = services.circuitos.map(pc => {
       const circuito = serviceState.circuitos.find(c => c.id === pc.circuitoId)!;
       const precioCircuito = serviceState.preciosCircuito.find(p => p.id === pc.precioCircuitoId);
       return { circuito, precioCircuito };
     });
     return calcularNeto(paquete, assignedAereos, assignedAlojamientos, assignedTraslados, assignedSeguros, assignedCircuitos);
   }, [services, serviceState, paquete]);
   ```

4. **Local markup state:** `const [localMarkup, setLocalMarkup] = useState(paquete.markup)`. Compute `computedVenta = calcularVenta(liveNeto, localMarkup)`.

5. **Save pricing handler:** Update paquete with new markup and computed values: `updatePaquete({ ...paquete, markup: localMarkup, netoCalculado: liveNeto, precioVenta: computedVenta })`. Show toast "Precios actualizados".

6. **Render:**
   - Glass Card container with `p-6`.
   - Section title: "Resumen de Precios" in h3.
   - `PriceDisplay` component with:
     - `neto={liveNeto}`
     - `markup={localMarkup}`
     - `venta={computedVenta}`
     - `onMarkupChange={canSeePricing.markup ? setLocalMarkup : undefined}`
     - `editable={canSeePricing.markup && canEdit}`
     - `size="lg"`
   - Below PriceDisplay: a breakdown section showing cost per service type (aereos total, alojamientos total, traslados total, seguros total, circuitos total) in a simple list with `formatCurrency()`. This helps the user understand where costs come from.
   - Save button: `<Button onClick={handleSave}>Guardar Precios</Button>` -- only shown if `canEdit`.

7. **VENDEDOR restrictions:** `canSeePricing` object from `useAuth()`:
   - If `!canSeePricing.neto`: hide the neto value and breakdown section.
   - If `!canSeePricing.markup`: hide markup and make PriceDisplay non-editable.
   - Venta is always visible (canSeePricing.venta is always true).

8. **All text in Spanish:** "Resumen de Precios", "Desglose de Costos", "Aereos", "Alojamientos", etc.
  </action>

**FotosTab.tsx:**

2. **Hooks:**

3. **Map fotos to ImageItem format:** The `ImageUploader` expects `ImageItem[]` with `{ id, url, alt }`. Map from `PaqueteFoto[]`:
   ```typescript
   const images: ImageItem[] = fotos.map(f => ({
     id: f.id,
     url: f.url,
     alt: f.alt,
   }));
   ```

4. **Handlers:**
   - `handleAdd(urls: string[])`: For each url, call `addFoto({ paqueteId: paquete.id, url, alt: \`Foto ${fotos.length + i + 1}\`, orden: fotos.length + i })`.

   - `handleReorder(reordered: ImageItem[])`: For each reordered image, compare orden with original. If changed, call `updateFoto({ ...originalFoto, orden: newIndex })` where originalFoto is found by id in the fotos array.

5. **Render:**
   - Glass Card container.
   - Section title: "Fotos del Paquete".
   - `<ImageUploader images={images} onAdd={canEdit ? handleAdd : undefined} onRemove={canEdit ? handleRemove : undefined} onReorder={canEdit ? handleReorder : undefined} maxImages={10} />`
   - If no photos and VENDEDOR: show "No hay fotos para este paquete".
