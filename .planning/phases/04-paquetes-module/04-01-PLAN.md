  <name>Task 1: Paquetes list page with glass table, search, and filter chips</name>
  <files>src/app/(admin)/paquetes/page.tsx</files>
  <action>
Replace the placeholder paquetes page with a full list page. This is a single large component file (~250-350 lines). Structure:

1. **Imports:** `useState`, `useMemo` from React. `useRouter` from `next/navigation`. `Plus`, `Edit`, `Copy`, `Trash2`, `Eye` from `lucide-react`. All glass UI components: `PageHeader`, `Button`, `SearchFilter` (+ `FilterChip` type), `Table/TableHeader/TableBody/TableRow/TableHead/TableCell`, `Badge`, `Modal/ModalHeader/ModalBody/ModalFooter`, `Pagination`. Provider hooks: `usePaquetes`, `usePackageActions`, `usePaqueteServices` from PackageProvider. `useTemporadas`, `useTiposPaquete` from CatalogProvider. `useAuth` from AuthProvider. `useAereos` from ServiceProvider. Utils: `formatCurrency`, `slugify` from `@/lib/utils`. Animations: `motion` from `motion/react`, `interactions` from `@/components/lib/animations`.

2. **Constants at module level:**
   - `estadoBadge` map: `{ ACTIVO: { variant: 'active', label: 'Activo' }, BORRADOR: { variant: 'draft', label: 'Borrador' }, INACTIVO: { variant: 'inactive', label: 'Inactivo' } } as const`
   - Items per page constant: 10

3. **Component state:**
   - `search: string` (search input value)
   - `activeFilters: string[]` (toggled filter chip values using namespaced format: `"temporada:{id}"`, `"estado:ACTIVO"`, `"tipo:{id}"`)
   - `currentPage: number` (pagination, starting at 1)
   - `deleteTarget: Paquete | null` (which paquete the delete modal is for, null = closed)
   - `isShaking: boolean` (shake animation trigger for delete confirmation)

4. **Derived data via useMemo:**
   - `filters: FilterChip[]` -- build from temporadas (label: temporada.nombre, value: `"temporada:{temporada.id}"`), estados (3 entries for ACTIVO/BORRADOR/INACTIVO), tipos (from useTiposPaquete). Each chip's `active` state derived from `activeFilters.includes(chip.value)`.
   - `filteredPaquetes` -- filter `paquetes` array with: (a) search match on `titulo` or `descripcion` (case-insensitive), (b) temporada filter via `startsWith("temporada:")`, (c) estado filter via `startsWith("estado:")`, (d) tipo filter via `startsWith("tipo:")`. Within each category: OR logic. Across categories: AND logic. Reset `currentPage` to 1 when filters change (use a useEffect or compute in handler).
   - `paginatedPaquetes` -- slice filteredPaquetes for current page
   - For each paquete row, derive `destino` by: looking up `usePaqueteServices(paquete.id).aereos[0]?.aereoId` in the aereos list from `useAereos()` to get the first aereo's `destino` field. If no aereo assigned, show `"--"`. NOTE: Since calling hooks in a loop is not allowed, create a `destinoMap: Record<string, string>` via useMemo that pre-computes destino for every paquete by reading `usePackageState().paqueteAereos` and `useAereos()`.

5. **Filter toggle handler:** `handleFilterToggle(value: string)` -- if value is in activeFilters, remove it; otherwise add it. Reset currentPage to 1.

6. **Clone handler:** `handleClone(id: string)` -- call `clonePaquete(id)` from `usePackageActions()`. Show a success toast: `useToast().addToast({ variant: 'success', title: 'Paquete clonado', description: 'Se creo una copia en estado Borrador' })`. Import `useToast` from Toast provider.

7. **Delete handler:** `handleConfirmDelete()` -- set `isShaking = true`, then after 400ms timeout: call `deletePaquete(deleteTarget.id)`, set `deleteTarget = null`, `isShaking = false`, show success toast. Use `setTimeout`.

8. **Render structure:**
   ```
   <PageHeader
     title="Paquetes"
     subtitle="Gestion de paquetes de viaje"
     action={canEdit && <Button onClick={() => router.push('/paquetes/nuevo')}><Plus /> Nuevo Paquete</Button>}
   />
   <SearchFilter
     searchValue={search}
     onSearchChange={setSearch}
     filters={filters}
     onFilterToggle={handleFilterToggle}
     placeholder="Buscar paquetes..."
   />
   <Table>
     <TableHeader>
       <TableRow>
         <TableHead>ID</TableHead>
         <TableHead>Titulo</TableHead>
         <TableHead>Destino</TableHead>
         <TableHead>Temporada</TableHead>
         <TableHead>Noches</TableHead>
         <TableHead>Estado</TableHead>
         {canSeePricing.neto && <TableHead>Neto</TableHead>}
         {canSeePricing.markup && <TableHead>Markup</TableHead>}
         <TableHead>Precio Venta</TableHead>
         {canEdit && <TableHead>Acciones</TableHead>}
       </TableRow>
     </TableHeader>
     <TableBody>
       {paginatedPaquetes.map(paquete => (
         <TableRow key={paquete.id} onClick={() => router.push(`/paquetes/${paquete.id}`)}>
           <TableCell>{paquete.id.slice(-4)}</TableCell>
           <TableCell>{paquete.titulo}</TableCell>
           <TableCell>{destinoMap[paquete.id] ?? '--'}</TableCell>
           <TableCell>{temporadaName lookup}</TableCell>
           <TableCell>{paquete.noches}</TableCell>
           <TableCell><Badge variant={estadoBadge[paquete.estado].variant}>{estadoBadge[paquete.estado].label}</Badge></TableCell>
           {canSeePricing.neto && <TableCell>{formatCurrency(paquete.netoCalculado)}</TableCell>}
           {canSeePricing.markup && <TableCell>{paquete.markup}%</TableCell>}
           <TableCell className="font-semibold">{formatCurrency(paquete.precioVenta)}</TableCell>
           {canEdit && <TableCell>action buttons (eye, edit, copy, trash)</TableCell>}
         </TableRow>
       ))}
     </TableBody>
   </Table>
   <Pagination ... />
   ```

   **IMPORTANT:** Use `paquete.id` as URL (not slugify), per research recommendation for slug collision avoidance. The detail page at `[slug]` will match by ID.

   **Action buttons in Acciones column:** Use small icon-only Button variants. Eye icon navigates to detail. Edit navigates to detail with `?tab=datos`. Copy calls `handleClone`. Trash opens the delete modal by setting `deleteTarget`. Wrap action button clicks with `e.stopPropagation()` to prevent row click navigation.

9. **Delete confirmation modal:**
   ```
   <Modal open={!!deleteTarget} onOpenChange={(open) => !open && setDeleteTarget(null)} size="sm">
     <ModalHeader>Eliminar Paquete</ModalHeader>
     <ModalBody>
       <motion.div
         animate={isShaking ? interactions.deleteShake.animate : {}}
         transition={interactions.deleteShake.transition}
       >
         <p>Esta seguro que desea eliminar "{deleteTarget?.titulo}"?</p>
         <p className="text-sm text-neutral-400 mt-2">Esta accion no se puede deshacer.</p>
       </motion.div>
     </ModalBody>
     <ModalFooter>
       <Button variant="ghost" onClick={() => setDeleteTarget(null)}>Cancelar</Button>
       <Button variant="danger" onClick={handleConfirmDelete}>Eliminar</Button>
     </ModalFooter>
   </Modal>
   ```

10. **VENDEDOR restrictions:** Wrap "Nuevo Paquete" button, action column header, action column cells, Neto column, and Markup column with `canEdit` / `canSeePricing` conditionals from `useAuth()`.

11. **All text in Spanish.** Labels: "Paquetes", "Gestion de paquetes de viaje", "Nuevo Paquete", "Buscar paquetes...", "Eliminar Paquete", "Cancelar", "Eliminar", column headers as specified.

12. **Temporada name lookup:** Create a `temporadaMap: Record<string, string>` via useMemo from `useTemporadas()` to map temporadaId -> nombre.

13. **Empty state:** If filteredPaquetes is empty, show a centered message: "No se encontraron paquetes" with a subtle icon.
