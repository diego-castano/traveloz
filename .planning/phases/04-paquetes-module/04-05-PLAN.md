  <name>Task 1: PublicacionTab with toggles, dates, estado, and etiquetas</name>
  <files>src/app/(admin)/paquetes/[slug]/_components/PublicacionTab.tsx</files>
  <action>
Create the Publicacion tab with all publication controls.

1. **Props:** `{ paquete: Paquete }`.

2. **Hooks:**
   - `usePackageActions()` for `updatePaquete`, `assignEtiqueta`, `removeEtiqueta`.
   - `usePaqueteServices(paquete.id)` -- use `.etiquetas` array for assigned etiquetas (PaqueteEtiqueta[]).
   - `useEtiquetas()` from CatalogProvider -- all available etiquetas for the active brand.
   - `useAuth()` for `canEdit`.
   - `useToast()` for feedback.

3. **Publicado toggle logic:** The `Paquete` type has NO `publicado` boolean field. Map as follows:
   - Toggle checked: `paquete.estado === 'ACTIVO'`
   - Toggle on: set `estado: 'ACTIVO'` via `updatePaquete`
   - Toggle off: set `estado: 'INACTIVO'` via `updatePaquete`
   - Show toast on change.

4. **Destacado toggle:** Directly maps to `paquete.destacado` boolean.
   - Toggle on: `updatePaquete({ ...paquete, destacado: true })`
   - Toggle off: `updatePaquete({ ...paquete, destacado: false })`

5. **Estado selector:** A `Select` dropdown with options: `BORRADOR`, `ACTIVO`, `INACTIVO`. Display labels: "Borrador", "Activo", "Inactivo". When estado changes, update via `updatePaquete` and sync the Publicado toggle display (since it derives from estado).

6. **Date pickers:** Two `DatePicker` components for `validezDesde` and `validezHasta`.
   - Initialize from `paquete.validezDesde` and `paquete.validezHasta` (ISO strings -- parse with `new Date()`).
   - On change: update paquete with new ISO string.
   - Label: "Valido Desde", "Valido Hasta".

7. **Etiquetas multi-select:**
   - Show currently assigned etiquetas as `Tag` pills with `removable` prop. Look up etiqueta name from `useEtiquetas()` by matching `paqueteEtiqueta.etiquetaId`.
   - "Agregar Etiqueta" section: A `Select` dropdown showing available etiquetas (those not already assigned). On select, call `assignEtiqueta({ paqueteId: paquete.id, etiquetaId: selectedId })`.
   - Remove: On Tag's remove click, call `removeEtiqueta(paqueteEtiqueta.id)` using the PaqueteEtiqueta junction record's id.

8. **Layout:** Glass Card container. Sections separated with subtle dividers:
   - Section 1: "Estado de Publicacion" -- Publicado toggle + Destacado toggle side by side, plus Estado selector below.
   - Section 2: "Periodo de Validez" -- Two DatePickers in a row.
   - Section 3: "Etiquetas" -- Assigned tags as pills + add dropdown.

9. **VENDEDOR restrictions:** All controls disabled/read-only when `!canEdit`. Toggles show current state but cannot be changed. Select and DatePicker are disabled. No "Agregar Etiqueta" dropdown. Remove buttons on tags are hidden.

10. **All text in Spanish:** "Estado de Publicacion", "Publicado", "Destacado", "Estado", "Borrador", "Activo", "Inactivo", "Periodo de Validez", "Valido Desde", "Valido Hasta", "Etiquetas", "Agregar Etiqueta", "Sin etiquetas asignadas".
