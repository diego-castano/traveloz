**Maps for view mode display (useMemo):**
```typescript
const proveedorMap = useMemo(() => {
  const m: Record<string, string> = {};
  for (const p of proveedores) { m[p.id] = p.nombre; }
  return m;
}, [proveedores]);

const ciudadMap = useMemo(() => {
  const m: Record<string, string> = {};
  for (const pais of paises) {
    for (const c of pais.ciudades) { m[c.id] = c.nombre; }
  }
  return m;

const paisMap = useMemo(() => {
```

```typescript
const [editingRowId, setEditingRowId] = useState<string | null>(null);
```

**Handlers:**

`handleStartEdit(t: Traslado)`: cancels any addingRow, sets `editingRowId(t.id)`, sets `draftRow({ nombre: t.nombre, tipo: t.tipo, paisId: t.paisId, ciudadId: t.ciudadId, proveedorId: t.proveedorId, precio: t.precio })`.

`handleCancelEdit()`: `setEditingRowId(null)`, `setDraftRow({})`.

`handleSaveEdit()`: calls `updateTraslado({ ...existingTraslado, ...draftRow })` where existingTraslado = `traslados.find(t => t.id === editingRowId)`. Guard: `if (!existing) return`. Toast 'Traslado actualizado'. Reset `editingRowId(null)` and `draftRow({})`.

`handleStartAdd()`: cancels any editingRow, sets `addingRow(true)`, sets `newRow({ nombre: '', tipo: 'REGULAR', paisId: '', ciudadId: '', proveedorId: '', precio: 0 })`.

`handleCancelAdd()`: `setAddingRow(false)`, `setNewRow({})`.

`handleSaveAdd()`: `createTraslado({ brandId: activeBrandId, nombre: newRow.nombre!, tipo: newRow.tipo!, paisId: newRow.paisId!, ciudadId: newRow.ciudadId!, proveedorId: newRow.proveedorId!, precio: newRow.precio! })`. Toast 'Traslado creado'. `setAddingRow(false)`, `setNewRow({})`.

`handleClone(t: Traslado)`: `createTraslado({ brandId: t.brandId, nombre: 'Copia de ' + t.nombre, tipo: t.tipo, paisId: t.paisId, ciudadId: t.ciudadId, proveedorId: t.proveedorId, precio: t.precio })`. Toast 'Traslado clonado'.

`handleConfirmDelete()`: `setIsShaking(true)`, after 400ms: `deleteTraslado(deleteTarget!.id)`, toast 'Traslado eliminado', `setDeleteTarget(null)`, `setIsShaking(false)`.

**Cascading selects in draftRow:** When `draftRow.paisId` changes, also reset `draftRow.ciudadId` to `''`:
```typescript
// In the paisId onValueChange handler:
setDraftRow(d => ({ ...d, paisId: newPaisId, ciudadId: '' }));
```
Same for `newRow` when `newRow.paisId` changes.

`ciudadOptions` for a given paisId:
```typescript
const draftCiudadOptions = useMemo(
  () => paises.find(p => p.id === draftRow.paisId)?.ciudades.map(c => ({ value: c.id, label: c.nombre })) ?? [],
  [paises, draftRow.paisId]
);

const newCiudadOptions = useMemo(
  () => paises.find(p => p.id === newRow.paisId)?.ciudades.map(c => ({ value: c.id, label: c.nombre })) ?? [],
  [paises, newRow.paisId]
);
```

**Table structure:** Use a plain `<table>` inside a `<Card>` (NOT the glass `<Table>` component — the Table component has motion.tbody stagger animations that conflict with frequent inline-edit re-renders per RESEARCH.md Pitfall 1 and anti-patterns).

Apply `glassMaterials.frosted` className to the wrapping Card or a div inside it for glass effect. Import `glassMaterials` from `@/components/lib/glass`.

Table `<thead>` with columns: ID, Nombre, Tipo, Pais, Ciudad, Proveedor, Precio, Acciones.

For each traslado row (`traslados.filter(search filter)`):

**View mode row** (editingRowId !== traslado.id):
- ID cell: `traslado.id.slice(-4)` with `text-xs text-neutral-400 font-mono`
- Nombre: `traslado.nombre`
- Tipo: Badge variant (teal for REGULAR, violet for PRIVADO) — or simple text
- Pais: `paisMap[traslado.paisId] ?? '--'`
- Ciudad: `ciudadMap[traslado.ciudadId] ?? '--'`
- Proveedor: `proveedorMap[traslado.proveedorId] ?? '--'`
- Precio: `formatCurrency(traslado.precio)`
- Acciones: `{canEdit && (<>Pencil onClick={handleStartEdit}, Copy onClick={handleClone}, Trash2 onClick={setDeleteTarget}</>)}`
  - Each action button: `e.stopPropagation()` not needed here (no row click navigation for Traslados)

**Edit mode row** (editingRowId === traslado.id):
- ID cell: same display (not editable)
- Nombre: `<input type="text" className={inlineInputClassName} value={draftRow.nombre ?? ''} onChange={...} />`
- Tipo: `<Select options={[{value:'REGULAR',label:'Regular'},{value:'PRIVADO',label:'Privado'}]} value={draftRow.tipo} onValueChange={v => setDraftRow(d => ({...d, tipo: v as TipoTraslado}))} />`
- Pais: `<Select options={paises.map(p => ({value:p.id,label:p.nombre}))} value={draftRow.paisId} onValueChange={v => setDraftRow(d => ({...d, paisId: v, ciudadId: ''}))} />`
- Ciudad: `<Select options={draftCiudadOptions} value={draftRow.ciudadId} onValueChange={v => setDraftRow(d => ({...d, ciudadId: v}))} />`
- Proveedor: `<Select options={proveedores.map(p => ({value:p.id,label:p.nombre}))} value={draftRow.proveedorId} onValueChange={v => setDraftRow(d => ({...d, proveedorId: v}))} />`
- Precio: `<input type="number" className={inlineInputClassName + ' text-right'} value={draftRow.precio ?? 0} onChange={e => setDraftRow(d => ({...d, precio: parseFloat(e.target.value) || 0}))} />`
- Acciones: Check (green, onClick=handleSaveEdit) + X (neutral, onClick=handleCancelEdit)

**Add row** (addingRow is true, rendered as last tbody row):
- ID cell: `<span className="text-xs text-neutral-300">nuevo</span>`
- Same input pattern as edit mode but using `newRow` state and `newCiudadOptions`
- Acciones: Check (handleSaveAdd) + X (handleCancelAdd)

**Search filter:** simple `.filter(t => t.nombre.toLowerCase().includes(search.toLowerCase()) || paisMap[t.paisId]?.toLowerCase().includes(...) || ciudadMap[t.ciudadId]?.toLowerCase().includes(...))` — applied before rendering rows.

**Layout:** PageHeader title "Traslados". Row above table: SearchFilter on left + "Nuevo Traslado" Button on right (inside `{canEdit && ...}`). Button onClick: `handleStartAdd`.
