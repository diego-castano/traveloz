```typescript
const [modalOpen, setModalOpen] = useState(false);
const [editTarget, setEditTarget] = useState<Proveedor | null>(null);
const [deleteTarget, setDeleteTarget] = useState<Proveedor | null>(null);
```

`handleOpenCreate`: sets `editTarget(null)`, resets form to empty strings, calls `setModalOpen(true)` — all synchronously, no useEffect.

`handleOpenEdit(p: Proveedor)`: sets `editTarget(p)`, sets form to `{ nombre: p.nombre, contacto: p.contacto ?? '', email: p.email ?? '', telefono: p.telefono ?? '', notas: p.notas ?? '' }`, calls `setModalOpen(true)`.

`handleSave`: if editTarget → `updateProveedor({ ...editTarget, ...form })` + toast 'Proveedor actualizado'; else → `createProveedor({ brandId: activeBrandId, ...form })` + toast 'Proveedor creado'. Then `setModalOpen(false)`.

`handleClone(p: Proveedor)`: `createProveedor({ brandId: p.brandId, nombre: 'Copia de ' + p.nombre, contacto: p.contacto ?? '', email: p.email ?? '', telefono: p.telefono ?? '', notas: p.notas ?? '' })` + toast 'Proveedor clonado'.

`handleConfirmDelete`: `setIsShaking(true)`, after 400ms: `deleteProveedor(deleteTarget.id)` (soft-delete), toast 'Proveedor eliminado', `setDeleteTarget(null)`, `setIsShaking(false)`.

Search filter: filter proveedores where `nombre` or `email` contains search term (case-insensitive).

Glass table columns: nombre, contacto, email, acciones (Eye/Pencil/Copy/Trash2 — Pencil+Copy+Trash inside `{canEdit && ...}`).

SearchFilter: `filters={[]}` `onFilterToggle={() => undefined}` (search-only, no chips — same fix as Aereos per 05-01-SUMMARY.md).

PageHeader title: "Proveedores". Add "Nuevo Proveedor" Button (inside `{canEdit && ...}`).

Create/Edit Modal (size="md"):
- ModalHeader title: editTarget ? 'Editar Proveedor' : 'Nuevo Proveedor'
- ModalBody: 5 Input fields — Nombre (required), Contacto, Email (type="email"), Telefono, Notas
- ModalFooter: Cancelar (ghost) + (editTarget ? 'Guardar' : 'Crear') Button

Delete modal: standard shake pattern with motion.div + `interactions.deleteShake` from `@/components/lib/animations`. Import `interactions` and `springs`. Use `animate={isShaking ? { x: [...interactions.deleteShake.animate.x] } : {}}`.

Pagination: Pagination component with `page`, `setPage`, `Math.ceil(filtered.length / PAGE_SIZE)`, slice paginated results.
