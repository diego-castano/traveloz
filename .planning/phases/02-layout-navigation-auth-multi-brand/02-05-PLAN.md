**Create `src/app/(admin)/layout.tsx`** ("use client"):

This is THE central layout that composes all Phase 2 components. Follow the exact pattern from the research (Pattern 4: Admin Layout Composition).

Imports:
- `{ Sidebar } from "@/components/layout/Sidebar"`
- `{ Topbar } from "@/components/layout/Topbar"`
- `{ AdminBackground } from "@/components/layout/AdminBackground"`
- `{ PageTransitionWrapper } from "@/components/layout/PageTransitionWrapper"`
- `{ useAuth } from "@/components/providers/AuthProvider"`
- `{ useRouter } from "next/navigation"`
- `{ useEffect } from "react"`

Auth redirect logic (research pitfall 2 - prevent flash of content):
```
const { isAuthenticated } = useAuth();
const router = useRouter();

useEffect(() => {
  if (!isAuthenticated) {
    router.push("/login");
  }
}, [isAuthenticated, router]);

// Return null while not authenticated to prevent flash
if (!isAuthenticated) {
  return null;
}
```

NOTE: Using useRouter + useEffect instead of redirect() because this is a client component. Returning null prevents any flash of admin content before redirect completes.

Layout structure (from design.json patterns.layout):
```jsx
<div className="flex h-screen overflow-hidden">
  <Sidebar />
  <div className="flex flex-1 flex-col overflow-hidden">
    <Topbar />
    <main className="relative flex-1 overflow-y-auto">
      <AdminBackground />
      <div className="relative z-[1] p-7">
        <PageTransitionWrapper>
          {children}
        </PageTransitionWrapper>
      </div>
    </main>
  </div>
</div>
```

Key structural details:
- Sidebar is fixed (it has position: fixed in its own component), so the flex layout only affects the main content area
- Main content area has padding: 28px (p-7 in Tailwind = 1.75rem = 28px) from design.json patterns.layout.contentPadding
- AdminBackground is fixed/absolute within main with z-0, content sits above at z-[1]
- PageTransitionWrapper wraps {children} to enable AnimatePresence transitions

**Update `src/app/page.tsx`:**

Replace the current placeholder with a redirect component:
```tsx
"use client";
import { useAuth } from "@/components/providers/AuthProvider";
import { useRouter } from "next/navigation";
import { useEffect } from "react";

export default function RootPage() {
  const { isAuthenticated } = useAuth();
  const router = useRouter();

  useEffect(() => {
    if (isAuthenticated) {
      router.replace("/paquetes"); // Default landing page for now (Dashboard is Phase 8)
    } else {
      router.replace("/login");
    }
  }, [isAuthenticated, router]);

  return null; // Brief blank while redirecting
}
```
