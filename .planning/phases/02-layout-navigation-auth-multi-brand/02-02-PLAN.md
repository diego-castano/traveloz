Also import Truck from lucide-react for proveedores.

**Component structure:**
1. State: `const [collapsed, setCollapsed] = useState(false)`
2. Get auth: `const { visibleModules } = useAuth()`
3. Get brand: `const { activeBrand } = useBrand()`
4. Get pathname: `const pathname = usePathname()`
5. Filter nav groups by role: filter items where `visibleModules.includes(item.id)`, then filter out empty groups
6. Detect active item: `pathname === item.href` or `pathname.startsWith(item.href + "/")` (but for "/" only exact match)

**Sidebar wrapper** -- use `motion.aside`:
- `animate={{ width: collapsed ? 64 : 252 }}`
- `transition={springs.gentle}` (locked decision: gentle spring for sidebar collapse)
- className: "fixed left-0 top-0 z-[100] h-screen flex-shrink-0 overflow-hidden flex flex-col"
- Style object (inline, NOT Tailwind for glass properties -- locked decision):
  - background: activeBrand.sidebarGradient
  - backdropFilter: `blur(${activeBrand.sidebarBlur})`
  - WebkitBackdropFilter: `blur(${activeBrand.sidebarBlur})` (Safari -- locked decision)
  - borderRight: "1px solid rgba(139,92,246,0.1)"

**Visual overlays (pointer-events: none):**
- Pulsing glow: div with className "animate-sidebar-glow" (keyframe already in tailwind.config.ts)
- Top glow: div with style `{ background: activeBrand.sidebarTopGlow }` and className "absolute inset-x-0 top-0 h-32 pointer-events-none"

**Logo area (top):**
- Fixed height ~60px, flex items-center, px-4
- When expanded: show brand logo text (activeBrand.name) in white
- When collapsed: show just a violet icon square
- Logo icon: 36x36 div with bg rgba(139,92,246,0.25), border 1px solid rgba(139,92,246,0.3), rounded-lg, box-shadow "0 0 16px rgba(139,92,246,0.2)"

**Nav section (scrollable middle):**
- className "flex-1 overflow-y-auto py-2 px-3"
- For each filtered group: render group label (only when expanded, text-xs uppercase tracking-widest, color rgba(255,255,255,0.3), mb-2 mt-4) then items
- Each nav item is a Link wrapped in a div:
  - className with cn() combining base styles + active/hover states
  - Base: "flex items-center gap-3 px-3 py-2.5 rounded-[10px] text-[13px] transition-all duration-200 cursor-pointer group"
  - Text color: rgba(255,255,255,0.55) default, rgba(255,255,255,0.9) on hover
  - Active state: bg rgba(139,92,246,0.2), border 1px solid rgba(139,92,246,0.15), boxShadow "inset 0 0 16px rgba(139,92,246,0.12), 0 0 12px rgba(139,92,246,0.08)", text white
  - Hover (non-active): bg rgba(255,255,255,0.08), translateX(2px)
  - Icon: size 18, strokeWidth 1.75 (from design.json iconography)
  - When collapsed: hide text, center icon, use Radix Tooltip to show label on hover (import { Tooltip } from "radix-ui")

**Collapse toggle button (bottom):**
- Absolute bottom, full width, h-12, flex center
- ChevronsLeft icon (when expanded) or ChevronsRight (when collapsed)
- Color: rgba(255,255,255,0.2), hover: rgba(255,255,255,0.5)
- onClick: setCollapsed(!collapsed)

**Dividers between groups:**
- 1px line with bg rgba(255,255,255,0.06), mx-3, my-1

IMPORTANT anti-patterns from research:
- Do NOT animate the element that has backdrop-filter directly for width changes -- motion.aside handles this correctly because the width animation is on the containing element, and the gradient is the background of that same element. This is fine since we are animating width (layout), not the filter.
- Do NOT hardcode violet tokens -- always read from activeBrand context
