**Create `src/components/layout/PageTransitionWrapper.tsx`** ("use client"):

Follow the EXACT FrozenRouter pattern from the research document. This is critical for page transitions in Next.js App Router.

Imports:
- `{ AnimatePresence, motion } from "motion/react"`
- `{ useSelectedLayoutSegment } from "next/navigation"`
- `{ LayoutRouterContext } from "next/dist/shared/lib/app-router-context.shared-runtime"` -- CRITICAL internal import, add comment documenting this is a non-public API that is stable in Next.js 14.x
- `{ useContext, useEffect, useRef, type ReactNode } from "react"`

**usePreviousValue hook** (internal helper):
```
function usePreviousValue<T>(value: T): T | undefined {
  const prevValue = useRef<T>();
  useEffect(() => {
    prevValue.current = value;
    return () => { prevValue.current = undefined; };
  });
  return prevValue.current;
}
```

**FrozenRouter component** (internal, not exported):
- Gets LayoutRouterContext via useContext
- Gets previous context via usePreviousValue
- Gets current and previous segment via useSelectedLayoutSegment + usePreviousValue
- Determines if segment changed: `segment !== prevSegment && segment !== undefined && prevSegment !== undefined`
- Renders LayoutRouterContext.Provider with value: changed ? prevContext : context
- Wraps {children}

**PageTransitionWrapper component** (exported):
- Gets segment via useSelectedLayoutSegment()
- Renders AnimatePresence with mode="wait" initial={false}
- Inside: motion.div keyed by segment with transition values from design.json animations.interactions.pageTransition:
  - initial: { opacity: 0, y: 12, filter: "blur(4px)" }
  - animate: { opacity: 1, y: 0, filter: "blur(0px)" }
  - exit: { opacity: 0, y: -8 }
  - transition: { duration: 0.3 }
- Inside motion.div: FrozenRouter wrapping {children}

CRITICAL NOTE in code comment: The LayoutRouterContext import from `next/dist/shared/lib/app-router-context.shared-runtime` is an internal Next.js API. It is stable across Next.js 14.x but may change in future major versions. This is the standard community pattern for AnimatePresence page transitions in the App Router.
