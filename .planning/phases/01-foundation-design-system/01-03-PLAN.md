<task type="auto">
  <name>Task 1: Build glass Table with dark header, row hover gradient, and stagger animation</name>
  <files>src/components/ui/Table.tsx</files>
  <action>
Create a composable Table system with these sub-components: Table, TableHeader, TableBody, TableRow, TableHead, TableCell.

1. **Table (wrapper)**:
   - Apply frosted glass material from glassMaterials.frosted as inline style
   - Classes: `w-full rounded-glass-lg overflow-hidden shadow-elevation-4`
   - Add top accent pseudo-element: `::before` with `linear-gradient(90deg, transparent 0%, rgba(139,92,246,0.12) 30%, rgba(59,191,173,0.12) 70%, transparent 100%)` as a thin 2px bar at top
   - Add `animate-breathe` class for subtle opacity breathing

2. **TableHeader**:
   - Inline style: `background: 'rgba(26,26,46,0.94)' , backdropFilter: 'blur(12px)', WebkitBackdropFilter: 'blur(12px)'`
   - No rounded corners (parent overflow hidden handles it)

3. **TableHead (th cells)**:
   - Classes: `px-4 py-3 text-left text-[11.5px] font-semibold uppercase tracking-[0.5px] text-white/90`
   - Special column styles via optional `variant` prop: id (font-mono text-xs), price (font-mono font-semibold), markup (font-mono text-xs text-orange-400)

4. **TableBody**: Use motion.tbody with stagger container variants from animations.ts:
   ```
   variants={{ hidden: {}, show: { transition: { staggerChildren: 0.04, delayChildren: 0.08 } } }}
   initial="hidden" animate="show"
   ```

5. **TableRow**: Use motion.tr with stagger item variants:
   - `variants={{ hidden: { opacity: 0, y: 8 }, show: { opacity: 1, y: 0 } }}`
   - Hover: Apply gradient background via Motion whileHover: `{ background: 'linear-gradient(90deg, rgba(139,92,246,0.03) 0%, rgba(59,191,173,0.04) 50%, rgba(139,92,246,0.02) 100%)' }`
   - Border bottom: `border-b border-neutral-150/50`
   - Selected state (via prop): `background: linear-gradient(90deg, rgba(59,191,173,0.06), rgba(139,92,246,0.03))`, left border inset 3px solid #8B5CF6

6. **TableCell**: `px-4 py-3 text-sm text-neutral-700`. Accept `variant` prop for special formatting (id: mono text-xs text-neutral-400, price: font-semibold mono, markup: mono text-xs text-orange-400).

7. **Props**: Table accepts `className`, all sub-components accept `className` + `children`. TableRow accepts optional `selected?: boolean`.

Use `React.forwardRef` on Table for composition.
