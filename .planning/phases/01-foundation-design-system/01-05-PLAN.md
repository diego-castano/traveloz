<task type="auto">
  <name>Task 1: Build Tabs with Radix + Motion layoutId gradient indicator</name>
  <files>src/components/ui/Tabs.tsx</files>
  <action>
Create Tabs using Radix Tabs primitive + Motion layoutId for animated indicator.

1. **Imports**: `import { Tabs as RadixTabs } from 'radix-ui'`, `import { motion } from 'motion/react'`, springs from animations.ts, cn from cn.ts.

2. **Tabs (root)**: Thin wrapper around RadixTabs.Root. Props: `value`, `onValueChange`, `defaultValue`, `className`, `children`.

3. **TabsList**: Wrapper around RadixTabs.List.
   - Classes: `relative flex border-b border-neutral-150/50`

4. **TabsTrigger**: Wrapper around RadixTabs.Trigger.
   - Classes: `relative px-4 py-2.5 text-[13.5px] font-medium transition-colors`
   - Inactive: `text-neutral-400 hover:text-neutral-600 hover:bg-neutral-100/40`
   - Active: `text-brand-teal-600 font-semibold` (apply via data-[state=active] selector)
   - **Animated indicator**: Conditionally render inside each trigger:
     ```tsx
     {isActive && (
       <motion.div
         layoutId="activeTab"
         className="absolute bottom-0 left-0 right-0 h-0.5 rounded-full"
         style={{ background: 'linear-gradient(90deg, #8B5CF6, #3BBFAD)' }}
         transition={springs.snappy}
       />
     )}
     ```
   - `isActive` determined by comparing `value` prop to parent's active value. Use Radix's `data-state` attribute or pass active value down via context.
   - IMPORTANT: The layoutId MUST be a unique string per Tabs instance. If multiple Tabs exist on same page, they need unique layoutId. Accept optional `layoutId` prop, default to `"activeTab"`.

5. **TabsContent**: Wrapper around RadixTabs.Content.
   - Classes: `mt-4 focus-visible:outline-none`
   - Optional: Add page transition animation (opacity fade) when switching content.

6. Props: Each sub-component accepts `className` for overrides.
  </action>
